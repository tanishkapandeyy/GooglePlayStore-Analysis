 Google Play Store Data Analysis Project

Project Overview

This project analyzes the Google Play Store dataset using Python and Jupyter Notebook. Multiple visualizations were developed to explore app performance, installs, ratings, reviews, and revenue across different categories.

The project consists of six analytical tasks, each involving specific filtering conditions, data transformations, visualization techniques, category translations, and time-based visibility restrictions.

 Tasks Completed

 Task 1 – Bubble Chart

* Visualized App Size vs Rating
* Bubble size represents Number of Installs
* Category-wise comparison
* Time-based visibility restriction implemented

 Task 2 – Interactive Choropleth Map

* Visualized global installs using Plotly Choropleth
* Displayed top 5 app categories
* Highlighted categories exceeding 1 million installs
* Interactive geographic visualization

 Task 3 – Time Series Line Chart

* Visualized install trends over time
* Applied category translations
* Highlighted periods with more than 20% month-over-month growth
* Implemented logarithmic scaling for readability

 Task 4 – Stacked Area Chart

* Visualized cumulative installs over time
* Applied multilingual category translations
* Highlighted months with significant install growth
* Implemented time-based visibility restriction

 Task 5 – Grouped Bar Chart

* Compared Average Rating and Total Reviews
* Displayed top 10 categories by installs
* Applied rating, size, and update-date filters

 Task 6 – Dual Axis Chart

* Compared Average Installs and Revenue
* Analyzed Free vs Paid applications
* Revenue calculated using:

Revenue = Installs × Price
